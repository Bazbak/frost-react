Redux - библиотека, которая отвечает за хранение и изменение глобального
состояния приложения.

Action - объект описывающий событие возникающее в приложении которое может
привести к изменению глобального состояния.
Объект action состоит из двух свойств:
1) type - тип возникающего события который является обычной строкой
записываемой в определенном формате category/name.
2) payload - данные которые должны быть переданы при возникновении события.

Action creator - функция отвечающая за создание объекта action.

Reducer - функция отвечающая за то как именно должно быть обновлено состояние.
Данная функция принимает в себя два параметра:
1) Состояние - актуальное состояние на момент вызова reducer функции.
2) Событие - объект action на основании которого reducer должен обновить
состояние.
Функция reducer в качестве результата должна вернуть уже измененное состояние.

Селектор - функция при помощи которой можно извлекать информацию из глобального
состояния.

dispatch - функция принимающая в себя объект action, применяется для обновления
глобального состояния.
